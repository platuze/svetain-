JavaScript užduotis:

  1. Į svetainę įdėti JavaScript kodą, kuris:
    • Yra funkcija
    • Funkcija savyje turi console.log(“SVEIKI, SKAITOTE CONSOLE.LOG?)
    • Funkcija iškviečiama atidarius puslapį

  2. Svetainėje turi būti mygtukas, tekstas ar kitas panašus elementas ant kurio paspaudus:
    • Pasikeičia jo spalva, dydis ar kitas stiliaus elementas (pasirinktinai). [PAPILDOMAI] Paspaudus ir vėl atsikeičia į prieš tai buvusį.
    • Kodas turi suveikti panaudojus event listener

  3. Į puslapį įdėti sekcija su nuotraukomis
